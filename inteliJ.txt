[ IDE ]
    IDE : 통합 개발 환경
    VSCODE는 프론트 환경에서 주로 사용
    JAVA 수업에는 INTELLIJ를 쓴다.

[ 인텔리제이 ]
    커뮤니티 / 윈도우 버전 다운로드
    JDK 다운로드 클릭 후 17버전으로 실행(공급업체는 아무거나)
    위치는 VSCODE 묶어둔 곳으로

[ 기본 설정 ]
    1. 모양 및 동작 - > 모양 테마 설정
    2. 에디터 -> 일반 -> ctrl 마우스 휠
    3. 플러그인 -> 외부 프로그램 설치 (프로젝트용)

[ 인텔리제이 프로젝트 구성 ]
    - 프로젝트명
        .idea (인텔리제이 설정파일 - 신경x)
        .out (자바 실행 결과파일 - 관리 대상x)
        .src (개발자가 작성한 코드 파일 - 관리 대상o)

[ 파일 만들기 ]
    1. [src]폴더 오른쪽 클릭
    2. [새로만들기]
        1) java 클래스
        2) 파일 ( 파일명.txt )
        3) 패키지 ( 폴더 )

[ 자바 클래스 파일 생성 ]
    1. [src]폴더 오른쪽 클릭
    2. [ 새로만들기 ] -> [ 패키지 ] -> ( 강의기준 day01~ )
    3. 특정폴더 폴더 오른쪽 클릭 - > [새로만들기] -> [java클래스]
        * 클래스 파일명은 첫 글자를 무조건 대문자로 시작.
        ex) Example1

[ 자바 파일 ]
    - Example1.java ( 파일 )

[ 인텔리제이 깃허브 연동 ]
    1. 인텔리제이 상단메뉴 -> [ 파일 ] -> [ 설정 ]
    2. 사이드메뉴 [ 버전관리 ] -> [ github ]
    3. [ 계정추가 ] -> github통해 -> 웹 깃허브 로그인 후 인증

[ 인텔리제이에서 깃허브 프로젝트 게시하기 ]
    1. 인텔리제이 상단메뉴 -> [ VCS ]
    2. [ Github 프로젝트 공유 ]
