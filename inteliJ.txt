[ 인텔리제이 ]
    커뮤니티 / 윈도우 버전 다운로드
    JDK 다운로드 클릭 후 17버전으로 실행(공급업체는 아무거나)

[ 기본 설정 ]
    1. 모양 및 동작 - > 모양 테마 설정
    2. 에디터 -> 일반 -> ctrl 마우스 휠
    3. 플러그인 -> 외부 프로그램 설치 (프로젝트용)

[ 인텔리제이 프로젝트 구성 ]
    - 프로젝트명
        .idea (인텔리제이 설정파일 - 신경x)
        .out (자바 실행 결과파일 - 관리 대상x)
        .src (개발자가 작성한 코드 파일 - 관리 대상o)

[ 파일 만들기 ]
    1. [src]폴더 오른쪽 클릭
    2. [새로만들기]
        1) java 클래스
        2) 파일 ( 파일명.txt )
        3) 패키지 ( 폴더 )

[ 자바 클래스 파일 생성 ]
    1. [src]폴더 오른쪽 클릭
    2. [ 새로만들기 ] -> [ 패키지 ] -> ( 강의기준 day01~ )
    3. 특정폴더 폴더 오른쪽 클릭 - > [새로만들기] -> [java클래스]
        * 클래스 파일명은 첫 글자를 무조건 대문자로 시작.
        ex) Example1

[ 자바 파일 ]
    - Example1.java ( 파일 )

[ 인텔리제이 깃허브 연동 ]
    1. 인텔리제이 상단메뉴 -> [ 파일 ] -> [ 설정 ]
    2. 사이드메뉴 [ 버전관리 ] -> [ github ]
    3. [ 계정추가 ] -> github통해 -> 웹 깃허브 로그인 후 인증

[ 인텔리제이 깃허브 커밋계정 정보 ]
    1. 인텔리제이 왼쪽 하단 메뉴 [ 터미널 : alt+F12 ]
    2. 터미널 상단 [v] 아이콘클릭 -> [ Git Bash ]
        git config --global user.name "KaFelLa92"
        git config --global user.email "choidj10@gmail.com"

[ 인텔리제이에서 깃허브 프로젝트 게시하기 ]
    1. 인텔리제이 상단메뉴 -> [ VCS ]
    2. [ Github 프로젝트 공유 ]
        - 저장소 이름 : 프로젝트명(*아무거나)
        - 원격 : (기본값)
    3. 최초 커밋 파일 (메시지 입력)

[ 커밋/푸시 하는 방법 ]
    1. 인텔리제이 왼쪽 메뉴 -> [ 커밋 ]
    2. 커밋할 파일들을 체크하고 하단에 메시지 작성
    3. [ 커밋 및 푸시 ] 클릭

[ 집 : 깃 프로젝트 클론/복제 ]
    1. 인텔리제이 첫 화면 -> [ 저장소 복제 ]
    * 왼쪽 사이드메뉴 -> [ github ] 계정없음 -> 클릭 후 계정 추가
    2. 복제할 저장소명을 검색 후 [복제]

[ 깃 저장소 삭제 ]
    1. 삭제할 저장소 github 웹페이지에서
    2. 저장소 [Settings] 메뉴에서 스크롤 내리고 Danzer Zone
    3. Delete this repository 클릭.
